# 🛒E-commerce - Frontend

A high-performance, responsive E-commerce web application built with **React** and **Vite**. This project features a modern user interface, secure authentication, and seamless product browsing experience.

## ✨ Key Features

- **User Authentication**: Secure Login and Registration system with real-time validation.
- **Form Management**: Robust form handling using **Formik** and schema-based validation with **Yup**.
- **Dynamic Routing**: Smooth navigation using **React Router DOM**.
- **State Management**: Global state handling via **React Context API** for user sessions.
- **API Integration**: Integrated with RouteMisr E-commerce REST API using **Axios**.
- **Responsive Design**: Fully mobile-responsive UI built with **Tailwind CSS**.
- **User Feedback**: Interactive success/error alerts and loading states.

## 🚀 Tech Stack

- **Core**: React.js (Hooks & Context)
- **Build Tool**: Vite
- **Styling**: Tailwind CSS / Vanilla CSS
- **Forms**: Formik & Yup
- **Networking**: Axios
- **Icons**: React Icons / Custom SVG components

## 🛠️ Project Structure

```text
src/
├── components/   # Reusable UI components (Login, Register, Nav, etc.)
├── context/      # Global state management (Auth Context)
├── assets/       # Images and static files
├── App.jsx       # Main application routing and layout
└── main.jsx      # Entry point
```

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```
